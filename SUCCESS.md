# AI OSS 실습기반 시험 성공 기준

시험 문항별 성공 기준을 체크박스 형태로 정리한 문서입니다.

## 문항 1. 협업 워크플로우 구성
- [ ] PR 링크 1개를 제출했다.
- [ ] 브랜치 전략 설명을 5문장 이내로 정리했다.
- [ ] 리뷰 코멘트 1개 이상을 남겼다.

## 문항 2. CI 파이프라인 구축 및 최적화
- [ ] 워크플로우 실행 링크 2개 이상을 남겼다.
- [ ] 실패 후 수정 성공 이력 1세트를 증빙했다.
- [ ] 최적화 전/후 실행 시간 비교표를 작성했다.

## 문항 3. Shift-left 테스트 실습
- [ ] 테스트 실행 로그를 남겼다.
- [ ] 실패 -> 수정 -> 성공 흐름이 보이는 커밋 2개 이상을 남겼다.
- [ ] 테스트 전략 설명을 5문장 이내로 정리했다.

## 문항 4. Feature Flag + TBD 적용
- [ ] OFF 상태 캡처 1개와 ON 상태 캡처 1개를 남겼다.
- [ ] 롤백 절차 3단계를 정리했다.
- [ ] 사용한 플래그 유형 1개와 선택 이유를 설명했다.

## 문항 5. 배포 및 운영 메트릭 연결
- [ ] 배포 결과 링크 또는 배포 성공 로그를 남겼다.
- [ ] 지표 수집 방식 요약표를 작성했다.

## 최종 제출 체크
- [ ] 최종 저장소 링크(Private)에 chunsejin을 협력자로 추가했다.
- [ ] 문항 1~5 관련 요구사항 답변을 포함했다.
- [ ] Actions 실행 링크 2개 이상을 포함했다.
- [ ] 배포 결과를 포함했다.
- [ ] 최종 요약 보고서를 포함했다.
